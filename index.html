How It Works

Model Pipeline

From raw patient review to 3-class sentiment prediction — a two-stage hybrid framework.

User Input
Patient submits a hospital review. HTML tags, URLs, special characters and contractions are cleaned and normalized.
Preprocessing
Contraction expansion, noise removal, and downsampling to 6,000 samples per class for a balanced 18,000-sample corpus.
BPE Tokenization
RoBERTa's Byte-Pair Encoding tokenizer decomposes text into subword tokens, padded or truncated to 128 tokens.
RoBERTa Encoding
Frozen roberta-base extracts 768-dimensional [CLS] token embeddings capturing deep bidirectional context. No fine-tuning required.
XGBoost Output
Regularized XGBoost classifier (multi:softprob) maps 768-D embeddings to Positive / Neutral / Negative with calibrated probabilities.
About the Model

Model Architecture

A Hybrid Transformer–Boosting Framework combining frozen RoBERTa embeddings with a regularized XGBoost multi-class classifier.

Architecture

RoBERTa-base + XGBoost — A two-stage decoupled design. RoBERTa (12 transformer layers, 768 hidden dimensions, 125M parameters) acts as a frozen feature extractor. Its 768-dimensional [CLS] token embeddings are fed directly into a regularized XGBoost multi-class classifier with multi:softprob objective.

Dataset

18,000 curated hospital reviews downsampled from 73,000 raw Yelp healthcare reviews, balanced across Positive, Neutral, and Negative categories — 6,000 samples each. Split 80/20 into 14,400 training and 3,600 testing samples using stratified sampling.

Training Configuration

  • Frozen RoBERTa
  • BPE Tokenization · max 128 tokens
  • XGBoost: 300 estimators, depth 3
  • L1 (α=1.0) + L2 (λ=2.0) Regularization
  • Learning Rate: 0.07 · Gamma: 0.5

Model Reliability

Achieves 86% accuracy and macro F1-score, outperforming TF-IDF + Logistic Regression, standalone XGBoost, fine-tuned BERT, and fine-tuned RoBERTa baselines. Macro-averaged AUC-ROC of 0.963 across all three classes. Highest confusion observed between Neutral and Negative — consistent with hedged language in mixed-sentiment healthcare reviews.
